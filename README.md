# Developer Snippets

##Flexbox 

Use Flexbox with Fallback:
Default: row
```sass
@include flexbox();
```


Specify Flexbox width:
Default: 50%
```sass
@include flexbox-width();
```


Specify Flexbox order:
Default: 1 up
```sass
@include flexbox-order();
```
